# Hong's AI Table Studio - 用户手册

## 1. 概览简介
欢迎使用 Hong's AI Table Studio！本应用旨在打造成您的灵动个人数据工作站。它能够让您像使用电子表格一样自如地组织信息，同时通过数据库级别的严格数据约束对您的表格进行专业降维管理。

## 2. 快速上手
当您打开该应用程序时，您将会看到主工作区被清晰划分为两个结构性区域：
1. **左侧边栏**: 您的空间表格导航窗格。在这里您可以创建、重命名、删除或复制不同的数据表。
2. **中间网格区**: 这是当前激活状态的表格面板，供您日常输入数据、定义列属性及变更不同的应用视图。

### 工作表管理
- **添加表格**: 点击侧边栏上方或列表底端的 `+` 或 "Add Table" (添加表格) 后，将为您生成一张全新的空白表单。
- **自定义图标**: 右键或双击您目标表格名字左侧的默认图标，以自定义其专属 Emoji 表情，这将显著增强您的视觉分类直观感。
- **重命名 / 复制 / 删除**: 鼠标指针悬停于侧边栏的一个具体表格名称上时，其右侧会浮现 "More" (更多/三个点) 菜单按钮。点击此按钮即可调出针对特定表格的管理交互指令。

## 3. 核心特性与功能
### 数据列字段类型
当您需要增添一个新列时，应用将允许您去深度定义该字段类型。这能极其严格地强制该列必须且仅接受代表了相关类型特征的信息输入：
- **Text (文本)**: 标准文本或记事内容。
- **Number (数字)**: 包含加减正负特性的数学数据。
- **1x Select (单选框) / Multi-Select (多选框)**: 这些属性可以为您创建一系列专属的下拉彩色标签，极其适用于诸如 "状态"、"优先级"、"个人分类标签" 这样的规范字段需求。
- **Formula (公式)**: 这是一个非常动态且具有 "只读" 特性的运算列，应用根据其余列中的值计算出最终结果并展示给您。

### 使用公式计算
当您创建一个 **Formula (公式)** 列后，您可以通过将想要引用的列名称存放在大括号 `{}` 的方式进行参数指代。
- **基础样例 (数学用法)**: `{单价} * {数量}`
- **进阶样例 (类 Excel 模式)**: 以等号 `=` 起手您的公式。例如：`={总库存} - {已售出}`。
如果您选择重命名一个正在被公式使用的列，请务必更新该公式本身，以防原有所绑定的对象指针因更名而丢失。

### 图像审阅与精准批注功能
当您在预览附件图片或 AI 生成的视觉效果素材图时：
- **大图沉浸审视**: 单击对应列的缩略图将其放大。 
- **打点备注 (Review Markers)**: 在图片的任意具象部位上 **直接双击鼠标** 即可打上一颗圆形的醒目「批注图钉」。 
- **分类处理状态**: 您可以轻松改变不同批注在当前的验收状态（"红色-待处理", "黄-已处理", 或 "绿-审核批准"）。 
- **跟帖互动沟通**: 各部门审核成员可集中点进任意批注针探讨互动，为后续 UI / 插画师的迭代保留精确追溯记录。 

### 桌面端极速本地图片直读
如果您是在 Windows / Mac 下当独立应用方式长驻使用该程序：
- **无需上传步骤**: 别再去进行冗长的寻找拖拽转储的操作了！只需将被选用的任意 C.D 盘内的本地原始绝对途径（例如 `C:\Users\Desktop\abc.png`）填写进带有图片属性性质的输入框内就行。
- 通过内部协议穿透能力，该界面环境将完美无感知地绕过网页端的沙盒隔离设定，瞬间把您存盘内的那张无缝链接并呈此刻眼前。 

### 拖拽重排功能
- **重新排列列**: 左键点击并按住一个列头开始向左或向右水平拖拽您的光标，可以重新排序整片网格列位置。
- **重新排布行**: 使用位于数据网格左侧尽头的每一行小抓手控件，去重新对当前记录条目进行物理换位交换。
- **重理工作表顺序**: 您随时可以在左栏随心拖拽各个不同表格的名字区块以符合逻辑重要等级。

## 4. 进阶功能：自动保存与备份
由于这是一款主要聚焦于客户端的纯网页应用，您的操作其实皆暂时寄存至当前浏览器进程中。如果想赋予其固若金汤避免丢失数据的防御机制，请参照：

1. 寻找应用侧上方的 **Settings / Preferences (设置页面齿轮)** 图标。
2. 在该弹出框中定位于 **Auto-Save Settings (自动备份功能)** 模块。
3. 把其设配到一个您可以随手访问的个人电脑本地专属文件夹内 (例如您的 "文档" 路径下)。 **注意: 出于当下严峻的浏览器层面隐形限制原则，诸如像 Windows 系里的核心 C: 盘深层嵌套或极少数的桌面特殊路径都属于高敏感受限地带，这极容易触发文件写入保护导致保存失灵。**
4. 当前应用一旦受到权限授意后，便会于幕后将持续变更的信息存做本地物理级的 `.json` 表内架构文档。即便系统在某个阶段为您强清了本地浏览器的高级缓存区，您亦可依仗并选择导入它以重溯当初状态。

## 5. 数据引入及外发
- **Import CSV (CSV 导入)**: 点击应用内相关的导入按键，去自动化批量转换一切兼容市面上多数的主流电子表为专属于应用内的数据结构。
- **Export CSV (CSV 导出)**: 若您需要离线报备及共享至他人，可通过相应按钮瞬间取下一份现阶段表格的 `.csv` 文件并于日后交递由微软 Microsoft Excel 或 Mac - Numbers 这样的跨平台办公软件解析查阅。

## 常见答疑
**问：为何我在分配自动同步存放设置项时总是弹出各种文件权限请求禁止以及保存失效的问题？**
答：为防止有恶意特性的跨维 Web 应用程序无节制侵占污染宿主操作系统级高危目录路径，市面上当下的主要浏览器厂商(如 Chrome 团队等)做出了严格限制判断。请移步并在操作系统内的用户文档区为当前软件独立开设出一个空白的专属文件夹（例如：此电脑 -> 我的文档 -> `TableStudioBaseData`），接着将上述纯净层级的地址选中到 Auto-Save 的捕获目标中即可。
